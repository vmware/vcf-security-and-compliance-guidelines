# Confidential Computing in VMware Cloud Foundation

Confidential Computing protects data in use by running workloads inside hardware-based trusted execution environments, where guest memory is encrypted and isolated using CPU features such as Intel Trust Domain Extensions (TDX) and AMD Secure Encrypted Virtualization-Secure Nested Paging (SEV-SNP). VMware Cloud Foundation (VCF) supports these technologies in newer releases, subject to version, hardware support, and licensing.
