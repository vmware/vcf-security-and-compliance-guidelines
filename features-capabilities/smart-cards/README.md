# Smart Card Authentication for VMware vCenter

> [!WARNING]
> The guides in this directory cover vCenter versions that have reached End of General Support.
> They are retained here solely as an archive. For current releases, consult the vSphere Security
> documentation for smart card and certificate-based authentication configuration.

This directory contains smart card authentication configuration guides for VMware vCenter 6.5, 6.7, and 7.0. These describe configuring certificate-based authentication, such as US Federal Common Access Cards (CAC), against those vCenter versions.

## Broadcom Resources

- [Smart Card Authentication Configuration Guide for VMware vCenter 6.5 (PDF)](https://raw.githubusercontent.com/vmware/vcf-security-and-compliance-guidelines/main/features-capabilities/smart-cards/VMware_vCenter_6.5_Smart_Card_Authentication_Configuration_Guide.pdf) -- direct download
- [Smart Card Authentication Configuration Guide for VMware vCenter 6.7 (PDF)](https://raw.githubusercontent.com/vmware/vcf-security-and-compliance-guidelines/main/features-capabilities/smart-cards/VMware_vCenter_6.7_Smart_Card_Authentication_Configuration_Guide.pdf) -- direct download
- [Smart Card Authentication Configuration Guide for VMware vCenter 7.0 (PDF)](https://raw.githubusercontent.com/vmware/vcf-security-and-compliance-guidelines/main/features-capabilities/smart-cards/VMware_vCenter_7.0_Smart_Card_Authentication_Configuration_Guide.pdf) -- direct download

To download these files from the GitHub web interface, use the "Download raw file" button on the file's page. Right-clicking the links above and choosing "Save Link As" also works.

## Current Guidance

For supported versions, smart card authentication is documented in the [vSphere Security documentation](https://techdocs.broadcom.com/us/en/vmware-cis/vsphere.html), and federated identity through an external identity provider is the recommended path for multifactor authentication in current releases. See also [Identity and Access Control in VMware Cloud Foundation](../identity-access-control/README.md).
