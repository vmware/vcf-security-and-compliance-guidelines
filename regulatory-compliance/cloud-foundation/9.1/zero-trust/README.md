# Zero Trust Architecture and VMware Cloud Foundation 9.1

This directory contains a whitepaper discussing zero trust architecture concepts and how VMware Cloud Foundation (VCF) 9.1 capabilities, along with separately licensed advanced services, can support an organization's zero trust efforts at the infrastructure layer. The discussion is grounded in common zero trust reference models, including the US Department of Defense Zero Trust capability framework and NIST SP 800-207.

- [**vcf-91-zero-trust-architecture.pdf**](vcf-91-zero-trust-architecture.pdf?raw=true) - the whitepaper

A related compliance companion guide for NIST SP 800-207 can be found in [../nist-800-207/](../nist-800-207/).
