# Regulatory Compliance for VMware Cloud Foundation 5.2

This directory contains the regulatory compliance materials published for VMware Cloud Foundation (VCF) 5.2, including the Compliance Kit, a DORA companion guide, and pointers to DISA STIG guidance. Each subdirectory has its own README with the download links.

These materials apply to VCF 5.2. The full set of current compliance companion guides is available at [https://brcm.tech/vcf-compliance](https://brcm.tech/vcf-compliance).

## Contents

- [VMware Cloud Foundation 5.2 Compliance Kit](compliance-kit/)
- [DISA STIG & STIG Readiness Guides](disa-stig/)
- [EU Digital Operational Resilience Act (DORA)](dora/)
