# Regulatory Compliance for VMware Cloud Foundation 4.5

This directory contains the regulatory compliance materials published for VMware Cloud Foundation (VCF) 4.5, including the Compliance Kit and pointers to DISA STIG guidance. Each subdirectory has its own README with the download links.

These materials are historical and apply to VCF 4.5. Current compliance companion guides are available at [https://brcm.tech/vcf-compliance](https://brcm.tech/vcf-compliance).
