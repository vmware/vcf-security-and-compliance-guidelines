# PCI DSS 3.2.1 — Product Applicability Guide

This directory contains the VMware Software-Defined Data Center (SDDC) Product Applicability Guide for version 3.2.1 of the Payment Card Industry Data Security Standard (PCI DSS), which governs the protection of cardholder data. It is provided as historical material published alongside VMware Cloud Foundation 4.2.

## Download

- [VMware PCI DSS SDDC Product Applicability Guide (PDF)](https://raw.githubusercontent.com/vmware/vcf-security-and-compliance-guidelines/main/regulatory-compliance/cloud-foundation/4.2/pci-dss-321/VMware_PCIDSS_SDDC_PAG-20210311.pdf) -- direct download

To download from the GitHub web interface, use the "Download raw file" button on the file's page. Right-clicking the link above and choosing "Save Link As" also works.

## Current Guidance

This is archived material for VMware Cloud Foundation 4.2. Current regulatory compliance companion guides, including PCI DSS 4.0.1, are available at [https://brcm.tech/vcf-compliance](https://brcm.tech/vcf-compliance).
