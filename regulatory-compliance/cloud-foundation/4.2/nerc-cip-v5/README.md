# NERC CIP v5 — Product Applicability Guide

This directory contains the VMware Product Applicability Guide for the North American Electric Reliability Corporation Critical Infrastructure Protection (NERC CIP) version 5 standards, which govern the security of the bulk electric system. It is provided as historical material published alongside VMware Cloud Foundation 4.2.

## Download

- [VMware Product Applicability Guide for NERC CIPv5 (PDF)](https://raw.githubusercontent.com/vmware/vcf-security-and-compliance-guidelines/main/regulatory-compliance/cloud-foundation/4.2/nerc-cip-v5/VMware%20Product%20Applicability%20Guide%20-%20NERC%20CIPv5%20-%202016-10-01.pdf) -- direct download

To download from the GitHub web interface, use the "Download raw file" button on the file's page. Right-clicking the link above and choosing "Save Link As" also works.

## Current Guidance

This is archived material for VMware Cloud Foundation 4.2. Current regulatory compliance companion guides, including NERC CIP, are available at [https://brcm.tech/vcf-compliance](https://brcm.tech/vcf-compliance).
