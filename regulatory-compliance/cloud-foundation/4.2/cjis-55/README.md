# CJIS Security Policy — Product Applicability Guide

This directory contains the VMware Product Applicability Guide for the FBI Criminal Justice Information Services (CJIS) Security Policy. The guide maps VMware capabilities to CJIS requirements and is provided as historical material published alongside VMware Cloud Foundation 4.2.

## Download

- [VMware End-User Computing Product Applicability Guide for CJIS (PDF)](https://raw.githubusercontent.com/vmware/vcf-security-and-compliance-guidelines/main/regulatory-compliance/cloud-foundation/4.2/cjis-55/vmware-euc-product-applicability-guide-cjis.pdf) -- direct download

To download from the GitHub web interface, use the "Download raw file" button on the file's page. Right-clicking the link above and choosing "Save Link As" also works.

## Current Guidance

This is archived material for VMware Cloud Foundation 4.2. Current regulatory compliance companion guides are available at [https://brcm.tech/vcf-compliance](https://brcm.tech/vcf-compliance).
