# HIPAA — Product Applicability Guide

This directory contains the VMware Product Applicability Guide for the US Health Insurance Portability and Accountability Act (HIPAA), which governs the protection of electronic protected health information (ePHI). It is provided as historical material published alongside VMware Cloud Foundation 4.2.

## Download

- [VMware Product Applicability Guide for HIPAA (PDF)](https://raw.githubusercontent.com/vmware/vcf-security-and-compliance-guidelines/main/regulatory-compliance/cloud-foundation/4.2/hipaa/VMware%20Product%20Applicability%20Guide%20-%20HIPAA%20-%202016-10-01.pdf) -- direct download

To download from the GitHub web interface, use the "Download raw file" button on the file's page. Right-clicking the link above and choosing "Save Link As" also works.

## Current Guidance

This is archived material for VMware Cloud Foundation 4.2. Current regulatory compliance companion guides are available at [https://brcm.tech/vcf-compliance](https://brcm.tech/vcf-compliance).
