# Cybersecurity Maturity Model Certification (CMMC)

This directory contains the VMware whitepaper on the Cybersecurity Maturity Model Certification (CMMC), the US Department of Defense framework for assessing the cybersecurity posture of contractors in the Defense Industrial Base. It is provided as historical material published alongside VMware Cloud Foundation 4.2.

## Download

- [VMware Whitepaper on Cybersecurity Maturity Model Certification (PDF)](https://raw.githubusercontent.com/vmware/vcf-security-and-compliance-guidelines/main/regulatory-compliance/cloud-foundation/4.2/cmmc/VMware%20Whitepaper%20on%20Cybersecurity%20Maturity%20Model%20Certification.pdf) -- direct download

To download from the GitHub web interface, use the "Download raw file" button on the file's page. Right-clicking the link above and choosing "Save Link As" also works.

This is archived material for VMware Cloud Foundation 4.2. Current regulatory compliance companion guides are available at [https://brcm.tech/vcf-compliance](https://brcm.tech/vcf-compliance).
