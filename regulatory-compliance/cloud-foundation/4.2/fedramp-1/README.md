# FedRAMP — Product Applicability Guide

This directory contains the VMware Product Applicability Guide for the Federal Risk and Authorization Management Program (FedRAMP), the US government program that standardizes security assessment and authorization for cloud products and services. It is provided as historical material published alongside VMware Cloud Foundation 4.2.

## Download

- [VMware Product Applicability Guide for FedRAMP v1.0 (PDF)](https://raw.githubusercontent.com/vmware/vcf-security-and-compliance-guidelines/main/regulatory-compliance/cloud-foundation/4.2/fedramp-1/vmware-product-applicability-guide-for-fedramp-v1-0.pdf) -- direct download

To download from the GitHub web interface, use the "Download raw file" button on the file's page. Right-clicking the link above and choosing "Save Link As" also works.

## Current Guidance

This is archived material for VMware Cloud Foundation 4.2. Current regulatory compliance companion guides are available at [https://brcm.tech/vcf-compliance](https://brcm.tech/vcf-compliance).
