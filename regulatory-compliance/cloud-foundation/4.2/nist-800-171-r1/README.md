# NIST SP 800-171 Revision 1 — Product Applicability Guide

This directory contains the VMware Software-Defined Data Center (SDDC) Product Applicability Guide for NIST Special Publication 800-171 Revision 1, which addresses the protection of Controlled Unclassified Information (CUI) in nonfederal systems. It is provided as historical material published alongside VMware Cloud Foundation 4.2.
