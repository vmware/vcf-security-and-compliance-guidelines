# Regulatory Compliance for VMware Cloud Foundation 4.2

This directory contains the regulatory compliance materials published for VMware Cloud Foundation (VCF) 4.2, including the Compliance Kit, Product Applicability Guides for individual frameworks, and pointers to DISA STIG guidance. Each subdirectory has its own README with the download links.

These materials are historical and apply to VCF 4.2. Current compliance companion guides are available at [https://brcm.tech/vcf-compliance](https://brcm.tech/vcf-compliance).
