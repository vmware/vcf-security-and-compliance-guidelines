# Regulatory Compliance Materials Filed Under VCF 5.1

This directory contains a DISA STIG pointer for VMware Cloud Foundation 5.1 along with two older compliance kits that predate it: the VMware Validated Design 5.1.1 Compliance Kits for NIST 800-53 and PCI DSS 3.2.1, published in 2019 and 2020. VMware Validated Design (VVD) was the predecessor to the standardized VMware Cloud Foundation architecture, and these kits are filed here because the VVD 5.1.1 designs correspond to that product generation. They are retained as an archive for organizations that deployed against those designs.

Each kit consists of multiple files meant to be used as a complete package: an introduction, a Product Applicability Guide, a configuration guide, an audit guide, and audit guide appendices. A zip of each complete kit is also present.

For current compliance guidance, see the [VCF 9.x compliance companion guides](../9.1/).
