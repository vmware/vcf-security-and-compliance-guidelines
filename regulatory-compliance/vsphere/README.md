# Regulatory Compliance for VMware vSphere

This directory contains regulatory compliance materials for standalone VMware vSphere, organized by version. Each version directory holds reference material documenting the platform's default security-relevant configuration, along with pointers to DISA STIG guidance.

For VMware Cloud Foundation compliance materials, see [cloud-foundation](../cloud-foundation/).

## Versions

- [VMware vSphere 8](8.0/)
- [VMware vSphere 7](7.0/)
- [VMware vSphere 6.7](6.7/)
- [VMware vSphere 6.5](6.5/)
